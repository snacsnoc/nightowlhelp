Sleep is hard and setting alarms is even more difficult-er. With two taps, you can set an alarm via SMS. Reliability is like, great!

How does it work?

Select a time, enter your phone number and voila! Save the contact (604-409-8733) with the same tone as your alarm.

Open source

All the code for SleepyPy is open source, view here (link goes here). SleepyPy uses Flask (Python) and Redis (via Redis Queue) with Twilio for SMS.

Why?

Setting alarms is something I don't like to do. I prefer to go to bed when I want and wake up whenever I feel best (which is impossible). Sleep is crucial for energy and overall well-being so it's important that I get quality sleep, so I built this tool. SMS is versatile and doesn't require any special app.
